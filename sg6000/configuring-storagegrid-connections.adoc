= 配置 StorageGRID 连接（ SG6000 ）
:icons: font
:imagesdir: ../media/


[role="lead"]
在将 StorageGRID 设备部署为 StorageGRID 系统中的存储节点之前，您必须配置设备与计划使用的网络之间的连接。您可以通过浏览到 StorageGRID 设备安装程序来配置网络连接，该安装程序预安装在 SG6000-CN 控制器（计算控制器）上。
