= 维护 SG5700 设备
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能需要升级 E2800 控制器上的 SANtricity 操作系统软件，更改 E5700SG 控制器的以太网链路配置，更换 E2800 控制器或 E5700SG 控制器或更换特定组件。本节中的过程假定设备已部署为 StorageGRID 系统中的存储节点。
