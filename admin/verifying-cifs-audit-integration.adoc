= 验证 CIFS 审核集成
:icons: font
:imagesdir: ../media/


[role="lead"]
审核共享为只读。日志文件可由计算机应用程序读取，验证不包括打开文件。我们认为，审核日志文件是否显示在 Windows 资源管理器窗口中已足够验证。验证连接后，关闭所有窗口。
