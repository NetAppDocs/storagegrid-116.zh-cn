= 支持的服务器证书类型
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID 系统支持使用 RSA 或 ECDSA （椭圆曲线数字签名算法）加密的自定义证书。

有关 StorageGRID 如何为 REST API 保护客户端连接的详细信息，请参见 xref:../s3/index.adoc[使用 S3] 或 xref:../swift/index.adoc[使用 Swift]。
