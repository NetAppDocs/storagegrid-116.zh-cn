= 示例： S3 对象删除
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
当 S3 客户端从存储节点（ LDR 服务）中删除对象时，系统会生成一条审核消息并将其保存到审核日志中。


NOTE: 在删除事务期间生成的审核消息并非都在以下示例中列出。仅列出与 S3 删除事务（ SDEL ）相关的那些。



== SDEL ： S3 删除

当客户端向 LDR 服务发送删除对象请求时，对象删除即开始。此消息包含用于删除对象的存储分段以及用于标识对象的 S3 密钥。
