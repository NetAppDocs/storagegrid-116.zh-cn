= 注册硬件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
注册设备硬件具有支持优势。

.步骤
. 找到机箱序列号。
+
您可以在装箱单上，确认电子邮件中或打开设备包装后在设备上找到此编号。

+
image::../media/appliance_label.gif[标签，包括序列号]

. 访问 NetApp 支持站点，网址为 http://["mysupport.netapp.com"^]。
. 确定是否需要注册硬件：
+
|===
| 如果您是 ... | 请按照以下步骤操作 ... 


 a| 
现有 NetApp 客户
 a| 
.. 使用您的用户名和密码登录。
.. 选择 * 产品 * > * 我的产品 * 。
.. 确认新序列号已列出。
.. 如果不是，请按照适用于新 NetApp 客户的说明进行操作。




 a| 
NetApp 新客户
 a| 
.. 单击 * 立即注册 * ，然后创建帐户。
.. 选择 * 产品 * > * 注册产品 * 。
.. 输入产品序列号和请求的详细信息。


注册获得批准后，您可以下载所需的任何软件。审批过程可能需要长达 24 小时。

|===
