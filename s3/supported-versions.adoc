= 支持的版本
:icons: font
:imagesdir: ../media/


[role="lead"]
StorageGRID 支持以下特定版本的 S3 和 HTTP 。

|===
| 项目 | version 


 a| 
S3 规范
 a| 
_Simple Storage Service API 参考 _ 2006-03-01



 a| 
HTTP
 a| 
1.1

有关 HTTP 的详细信息，请参见 HTTP/1.1 （ RFC 7230-35 ）。

* 注 * ： StorageGRID 不支持 HTTP/1.1 管道化。
