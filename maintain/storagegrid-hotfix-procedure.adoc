= StorageGRID 热修补程序操作步骤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果检测到软件问题并在功能版本之间得到解决，则可能需要将修补程序应用于 StorageGRID 系统。

StorageGRID 修补程序包含在功能或修补程序版本之外进行的软件更改。未来版本也会进行同样的更改。此外，每个热修补程序版本都包含此功能或修补程序版本中所有以前的修补程序的汇总。
