= 从系统驱动器故障中恢复
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果基于软件的存储节点上的系统驱动器发生故障，则此存储节点不可用于 StorageGRID 系统。您必须完成一组特定任务才能从系统驱动器故障中恢复。

.关于此任务
使用此操作步骤 从基于软件的存储节点上的系统驱动器故障中恢复。此操作步骤 包括在任何存储卷同样出现故障或无法重新挂载时应遵循的步骤。


NOTE: 仅限此基于 操作步骤 适用场景 软件的存储节点。要恢复设备存储节点，您必须遵循其他操作步骤 。
